# RESTful API Test to Supero with Spring Boot, Spring Data REST and MySQL

## Simple project based in KISS principle. Reference: https://en.wikipedia.org/wiki/KISS_principle


## Prerequisites
- JDK 1.8 
- Maven 4
- MySQL 5.6

## Stack
- Spring Boot
- Spring Data JPA
- Spring Data REST
- MySQL

## Run
`mvn clean spring-boot:run`
